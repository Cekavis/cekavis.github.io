「SPOJ QTREE」Query on a tree

SPOJ QTREE

题意

你有一棵\(n\)个点的树，边有权，有两种操作

  • CHANGE x y，表示修改第\(x\)条边的边权为\(y\)

  • QUERY x y，表示询问点\(x\)到\(y\)路径上的边权最大值
